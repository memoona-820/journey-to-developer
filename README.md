# 🎬 Journey to Developer

A personal multimedia portfolio documenting one person's journey — from childhood dreams to becoming a Full Stack Developer.

Built as a beginner project for the **Sigma Web Development Course** and redesigned with a modern portfolio aesthetic.

---

## 📖 About

This website tells a story in **6 chapters** through video. It traces the path from early career dreams, through moments of doubt and change, to ultimately discovering a passion for programming, MERN Stack development, and AI.

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

---

## ✨ Features

- 📹 6 video chapters with descriptions
- 🎵 6 audio tracks (Urdu songs)
- Responsive grid layout
- Scroll-triggered fade-in animations
- Dark hero section with modern typography
- Mobile-friendly design

---

## 📁 Project Structure

```
/
├── index.html      # Main page
├── style.css       # Styles
├── script.js       # Animations
├── 1.mp4 – 6.mp4  # Video chapters
└── 1.mp3 – 6.mp3  # Audio tracks
```

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/memoona-820/journey-to-developer.git
   ```
2. Open `index.html` in your browser — no setup needed.
